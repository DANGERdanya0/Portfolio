# Документация по Админ-Панели Портфолио

## 1. Введение

Эта документация описывает функционал админ-панели для сайта-портфолио преподавателя. Админ-панель предназначена для управления всем динамическим контентом на сайте, позволяя нескольким пользователям легко адаптировать и персонализировать свои портфолио без необходимости изменять код.

---

## 2. Доступ к Админ-Панели

Для входа в режим редактирования сайта предусмотрена специальная кнопка.

- **Расположение кнопки**: Кнопка с иконкой шестеренки (⚙️) будет расположена в левом нижнем углу сайдбара.
- **Активация**: При нажатии на эту кнопку поверх основного сайта будет открываться модальное окно.
- **Модальное окно**: Окно админ-панели будет содержать все настройки, описанные в этой документации. Его можно будет закрыть, вернувшись к просмотру сайта.

---

## 3. Общие Настройки


Эти настройки влияют на сайт в целом.

| Настройка | Поле в Админ-Панели | Описание |
| :--- | :--- | :--- |
| **ФИО Преподавателя** | Текстовое поле "ФИО" | Изменяет имя, которое отображается в сайдбаре и в разделе "Визитка". |
| **Управление Блоками** | Группа переключателей (toggles) | Позволяет включать или отключать отображение целых разделов (блоков) на сайте. |
| **Выбор темы оформления** | Радио-кнопки (выбор одного) | Позволяет изменить визуальный стиль всего сайта. |

### Список Управляемых Блоков

- [ ] `🏠 Главная`
- [ ] `👤 Визитка`
- [ ] `📚 Методическая копилка`
- [ ] `📊 Результаты`
- [ ] `📰 Блог`
- [ ] `🏆 Достижения`
- [ ] `👨‍🎓 Обучающимся`
- [ ] `📸 Галерея`
- [ ] `📬 Обратная связь`

### Варианты тем

1.  **Стандартная (по умолчанию)**
    *   Светлая тема с сине-фиолетовым градиентом, как на данный момент.

2.  **Темная**
    *   Элегантная темная тема с акцентными цветами, комфортная для просмотра в условиях низкой освещенности. Основной фон - темно-серый, текст - светлый.

3.  **Минимализм**
    *   Светлая, чистая тема с акцентом на типографику. Убраны градиенты и тени, используется простая палитра (белый, серый, черный) с одним ярким акцентным цветом (например, синим).

## 3. Управление Контентом Разделов

Каждый раздел сайта имеет свои уникальные управляемые элементы.

### 🏠 Главная (`#home`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Контактная информация** | Форма с текстовыми полями | Позволяет редактировать ФИО, место работы, образование и email. |
| **Карточки "Основные разделы"**| Список карточек с кнопками | Добавление, редактирование, удаление и изменение порядка карточек с новостями и анонсами. Каждая карточка имеет заголовок, текст и дату. |

### 👤 Визитка (`#business-card`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Фотография** | Загрузчик изображений | Загрузка/замена фотографии преподавателя. |
| **Персональная информация**| Форма с текстовыми полями | Редактирование должности, места работы, ученой степени, звания и т.д. |
| **Навыки** | Редактор тегов | Добавление и удаление тегов с названиями преподаваемых дисциплин. |
| **Награды и достижения** | Список карточек с кнопками | Добавление, редактирование и удаление карточек с наградами. Каждая карточка имеет название и возможность прикрепить файл (PDF). |

### 📚 Методическая копилка (`#methodical`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Рабочие программы** | Таблица с элементами управления | Добавление, редактирование и удаление строк в таблице. Каждая строка содержит название, курс, специальность и файл. |
| **Методические разработки**| Список карточек с кнопками | Управление карточками разработок (название, описание, количество файлов/тип). |
| **Публикации** | Список карточек с кнопками | Управление карточками публикаций (название, год, прикрепленный файл). |

### 📊 Результаты (`#results`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Таблицы успеваемости** | Редактор таблиц | Позволяет добавлять, изменять и удалять строки в таблицах "Мониторинг успеваемости", "Сравнительный анализ" и "Участие в чемпионатах". |

### 📰 Блог (`#blog`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Новостные посты** | Список постов с кнопками | Создание, редактирование и удаление новостных постов. Каждый пост содержит заголовок, текст и дату. |

### 🏆 Достижения (`#extracurricular`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Достижения обучающихся**| Редактор таблиц | Добавление, редактирование и удаление строк в таблице с информацией о студентах, их достижениях и датах. |
| **Фото с мероприятий** | Управление галереей | Добавление и удаление изображений с подписями. |

### 👨‍🎓 Обучающимся (`#students`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Карточки с материалами**| Список карточек с кнопками | Управление карточками в разделах "Конспекты", "Онлайн-ресурсы", "Расписание", "Анонсы". |
| **Видеоуроки** | Список карточек с кнопками | Добавление/удаление видеоуроков (карточка с названием и, возможно, ссылкой на видео). |

### 📸 Галерея (`#gallery`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Изображения** | Управление галереей | Загрузка, удаление и изменение порядка изображений в основной фотогалерее. |

### 📬 Обратная связь (`#guestbook`)

| Элемент | Управление | Описание |
| :--- | :--- | :--- |
| **Записи гостевой книги**| Список записей с кнопками | Просмотр, удаление и, возможно, скрытие/публикация отзывов, оставленных пользователями. |
